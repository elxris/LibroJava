LibroJava
=========

Es un proyecto para solucionar los ejercicios del libro de "Java Como programar", 7° Edición. Este proyecto está en español. Recordar que esta es sólo una manera de resolver un problema, pueden existir otras mejores.

Cualquier daño que el código aquí presente pueda causar es responsabilidad del que lo utiliza.

Los exorto a hacer un fork al proyecto y participar. :D
